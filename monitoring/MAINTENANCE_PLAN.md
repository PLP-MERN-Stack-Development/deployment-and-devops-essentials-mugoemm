# Maintenance Plan

- **Regular Updates:**
  - Keep dependencies up to date (use `npm outdated` and `npm update` regularly)
  - Monitor for security advisories
- **Database Backups:**
  - Enable automated backups in MongoDB Atlas
  - Test restore process periodically
- **Monitoring:**
  - Review Sentry and uptime dashboards weekly
  - Set up alerting for critical errors and downtime
- **Documentation:**
  - Update deployment and rollback instructions after every major change
- **Incident Response:**
  - Document steps for rolling back deployments (use GitHub revert and redeploy)
  - Keep contact info for all team members and service providers
